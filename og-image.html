📸 OG 이미지 생성 방법

방법 1. 스크린샷 (가장 쉬움)
1. 이 파일을 브라우저에서 엽니다 (og-image.html)
2. 개발자도구(F12) → Elements → <div class="og-card"> 우클릭 → "Capture node screenshot"
3. 저장된 PNG 파일명을 og-image.png로 바꿔 루트 폴더에 저장

방법 2. Puppeteer / Playwright (자동화)
npx playwright screenshot --viewport-size=1200,630 --element='.og-card' og-image.html og-image.png

방법 3. macOS 스크린샷
1. 브라우저 창을 1200px 이상으로 키우기
2. Cmd + Shift + 4 후 파란 카드 영역만 드래그
3. 프리뷰에서 1200×630으로 리사이즈 → og-image.png로 저장

💡 디자인 수정하고 싶다면: 이 HTML 파일의 텍스트/색상을 직접 편집하세요.
