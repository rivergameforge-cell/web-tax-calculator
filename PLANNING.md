## 서비스 개요

| 항목 | 내용 |
|------|------|
| 서비스명 | 한국 세금 계산기 |
| 목적 | 한국 주요 세금을 누구나 쉽게 계산할 수 있는 무료 웹 서비스 |

### ✅ MVP 1 — 핵심 계산기 (완료)

> 가장 조회 수가 높을 계산기 우선 구현

**완료된 작업:**
- 🏦 **대출금 계산기** — 원리금균등 / 원금균등 / 만기일시 상환, 월별 상환 스케줄
- 🏠 **취득세** — 주택/비주택, 가격구간별 세율, 생애최초/출산공제
- 🏠 **양도소득세** — 누진세율, 1세대1주택 비과세, 장기보유특별공제
- 🏠 **부동산 복비** — 매매/전세/월세, 주택/비주택, VAT 포함
- 🚗 **자동차세** — CC별 세율, 차령감경, 선납할인
- 📈 **해외주식 양도소득세** — 250만원 공제, 22% 세율
- 📈 **증권거래세** — 코스피/코스닥/코넥스/비상장 2026년 세율

### 🚧 MVP 2 — 부동산 카테고리 완성

> 부동산 관련 나머지 7개 계산기 구현
> 예상 작업량: 중~대

**구현 목표:**

#### 종합부동산세 (real-estate/comprehensive)
- 주택 공시가격 합산 → 공정시장가액비율 적용
- 기본공제 9억원 (1주택자 12억원)
- 누진세율 (0.5%~5%), 세부담상한 적용
- 농특세 20% 추가

#### 재산세 (real-estate/property-tax)
- 토지/건물/주택 유형별 세율
- 주택: 6천만 이하 0.1% ~ 3억 초과 0.4% 누진
- 도시지역분 0.14% 추가, 지방교육세 20%

#### 임대소득세 (real-estate/rental-income)
- 2천만원 이하: 분리과세 14% 선택 또는 종합과세
- 2천만원 초과: 종합소득세 합산
- 필요경비율 (등록임대 60%, 미등록 50%)

#### 상속세 (real-estate/inheritance)
- 누진세율 10%~50%
- 기초공제 2억, 일괄공제 5억, 배우자공제
- 금융재산 공제, 동거주택 공제

#### 증여세 (real-estate/gift)
- 누진세율 10%~50%
- 증여재산공제 (배우자 6억, 직계존속 5천만, 기타)
- 10년 내 동일인 합산

#### 인지세 (real-estate/stamp)
- 부동산 거래금액별 정액세
- 1천만원 이하 50,000원 ~ 10억 초과 350,000원

#### 등록면허세 (real-estate/registration)
- 소유권보존/이전/설정 유형별
- 부동산 가액 × 0.8% (이전), 주택 감면 적용

### 🔲 MVP 3 — 소득 카테고리 완성

> 직장인·사업자 관련 계산기, 조회 수 잠재력 높음
> 예상 작업량: 대

**구현 목표:**

#### 근로소득세·연말정산 (income/employment)
- 급여 입력 → 근로소득공제 자동 적용
- 인적공제 (본인/배우자/부양가족)
- 특별공제 (보험료/의료비/교육비/주택)
- 세액공제 (근로소득세액공제, 자녀세액공제)
- 예상 연말정산 환급/추가납부 계산

#### 사업소득세 (income/business)
- 수입금액 → 업종별 경비율 적용
- 종합소득세 누진 적용

#### 종합소득세 (income/comprehensive)
- 6개 소득 합산 (근로/사업/이자/배당/연금/기타)
- 6%~45% 누진세율 (2026년)
- 소득공제 항목 입력

#### 이자·배당소득세 (income/interest-dividend)
- 2천만원 이하: 원천징수 14% (이자), 14% (배당)
- 금융소득종합과세 (2천만원 초과 시 종합과세 합산)
- 배당세액공제 11%

#### 퇴직소득세 (income/retirement)
- 근속연수공제, 환산급여공제 단계적 적용
- 누진세율 후 근속연수로 재환산
- 실효세율 계산

#### 부가가치세 (income/vat)
- 일반과세자: 매출세액 - 매입세액
- 간이과세자: 공급대가 × 업종별 부가가치율 × 10%
- 과세기간 (1기/2기) 선택

#### 해외 구매 관세 (income/customs)
- 물품가격 + 운임·보험료 → 과세가격
- 품목별 관세율 (의류 13%, 전자제품 0~8% 등)
- 부가세 10%, 개별소비세 (해당 품목), 150달러 면세 기준

### 🔲 MVP 4 — 자동차·주식/금융 완성

> 나머지 자동차 2종 + 주식 2종
> 예상 작업량: 소~중

**구현 목표:**

#### 자동차 취득세 (vehicle/acquisition)
- 취득가액 × 7% (기본)
- 경차 4%, 하이브리드 4%, 전기차 취득세 감면
- 영업용/비영업용 구분

#### 자동차 개별소비세 (vehicle/excise)
- 출고가 기준 5% (개별소비세)
- 교육세 30%, 부가가치세 10% 추가
- 하이브리드/전기차 감면

#### 국내주식 양도소득세 (stocks/domestic)
- 대주주 기준 (종목당 10억 이상)
- 중소기업 10% / 일반 20~25% / 1년 미만 30%
- 기본공제 250만원

#### 배당소득세 (stocks/dividend)
- 국내주식 배당: 14% 원천징수
- 금융소득종합과세 판단 (2천만원 초과)
- 해외주식 배당: 15% 외국납부세액공제

### 🔲 MVP 5 — 기타 카테고리 & 마무리

> 안내형 페이지 + 서비스 완성도 향상
> 예상 작업량: 소

**구현 목표:**

#### 주민세 (other/resident)
- 개인균등분: 연 1만원
- 사업소분: 사업장 면적 기준
- 종업원분: 급여총액 × 0.5%

#### 안내형 페이지 (유류세/관세/개별소비세/주세/담배소비세/레저세)
- 계산기 대신 **세율 안내 카드** 형식
- 현행 세율 표, 계산 방식 설명

- 웰컴 화면 개선 (인기 계산기 바로가기)

- Google AdSense 실제 publisher ID 연결
- 사이트맵 생성
- 404 처리 (잘못된 해시 → 웰컴 화면)
- 공유하기 버튼 (카카오톡/링크 복사)
- 인쇄 기능 (`window.print()`)

## 개발 컨벤션

### 파일 네이밍
- 계산기 JS: `camelCase` → `CalcXxx` 네임스페이스
- CSS 클래스: `kebab-case`
- ID: `kebab-case` (예: `view-real-estate-acquisition`)

### 계산 로직
- 세율 상수는 파일 상단에 `const` 객체로 선언
- 계산 함수는 순수 함수 (DOM 접근 없음)
- 포맷팅은 반드시 `UI.fmtWon()` / `UI.fmtNum()` 사용

### 뷰 HTML 구조 패턴
```html
<div id="view-category-name" class="calculator-view">
  <div class="calc-container">
    <!-- 입력 패널 -->
    <div class="calc-panel input-panel">
      <div class="panel-header"><h2>계산기 이름</h2></div>
      <div class="panel-body">
        <!-- form groups -->
      </div>
      <button class="btn-calc" id="btn-xxx-calc">계산하기</button>
    </div>
    <!-- 결과 패널 -->
    <div class="calc-panel result-panel" id="result-xxx">
      <!-- 결과 카드 -->
    </div>
  </div>
</div>
```
